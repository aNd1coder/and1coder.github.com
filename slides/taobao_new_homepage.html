典型的一次 DNS 解析耗费 20-120 毫秒，减少 DNS 解析数是个优化的方式，而能够缩减 DNS 解析的时间也是有经济效益的事情。

iconfont

  • 优点
  • 缺点
  • 使用
  • 资源

优点

»

  • 可以设置颜色，大小，阴影
  • 字体文件很小，20-30kb左右

缺点

  • 只适用于纯色扁平的图形（最多在高端浏览器比如webkit上支持些渐变和蒙板）
  • 字体跨域（IE9+，FF）

    根据Request Headers中的Origin来配置Reponse Headers中的Access-Control-Allow-Origin

    http://km.oa.com/group/19251/articles/show/129688
  • 维护难度增大，如icon有变动则需要更多时间进行调整，中间涉及较多工具，不宜大规模使用

使用


                        @font-face {
                            font-family: iconfont;  /*名称可跟字体文件名不同*/
                            src: url(http://g.tbcdn.cn/tb/icon-font/1.0.6/iconfont.eot);   /* IE9 Compat Modes */
                            src: url(http://g.tbcdn.cn/tb/icon-font/1.0.6/iconfont.eot?#iefix) format("embedded-opentype"),   /* IE6-IE8 */
                            url(http://g.tbcdn.cn/tb/icon-font/1.0.6/iconfont.woff) format("woff"),   /* Modern Browsers */
                            url(http://g.tbcdn.cn/tb/icon-font/1.0.6/iconfont.ttf) format("truetype"),   /* Safari, Android, iOS */
                            url(http://g.tbcdn.cn/tb/icon-font/1.0.6/iconfont.svg#uxiconfont) format("svg")  /* Legacy iOS */
                        }
                

                        .iconfont {
                            font-family: iconfont;
                            font-size: 12px;
                            font-style: normal;
                            -webkit-font-smoothing: antialiased;
                            -moz-osx-font-smoothing: grayscale;
                        }
